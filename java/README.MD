# Java Utilities (Console Demos)

These Java programs simulate backend/business logic.

## Files
- Main.java — tests Java setup and uses Calculator
- Calculator.java — business math utilities (discount, tax, split)
- LeadCollector.java — console app storing leads (demo backend)

## How to run
Open a terminal in the `java` folder:

javac Main.java
java Main

javac LeadCollector.java
java LeadCollector
